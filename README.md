# car_rental
The Car Rental Management System is a desktop-based  application developed using Python and Tkinter

****
Tools & Technologies Used**** 
1. Python 
Main programming language used for logic and structure. 
2. Tkinter 
Pythonâ€™s standard GUI (Graphical User Interface) library. 
Used for building the user interface, windows, buttons, forms, and 
tables. 
3. ttk (Themed Tkinter Widgets) 
For advanced widgets like Treeview (used to display tables for 
customers, cars, rentals, and payments). 
4. datetime module 
Handles and validates rental start and end dates. 
5. messagebox (from tkinter) 
Provides user feedback via pop-up messages (e.g., success, error, info 
alerts). 
6. Custom in-memory data structures 
Lists and dictionaries are used to store and manage Customers, Cars, 
Rentals, and Payments.
